First Post! Introduction

Posted May 26, 2015

Hello! Can you hear me? Testing!

Ahem. Now that we're all set, here's a brief explanation:

A long time ago, in a galaxy far away...

No! That's a different story. This one really only started about a month ago, when I actively started coding with HTML and JavaScript. I learned HTML in middle school, but at that time HTML was pretty much the only thing used in web design. Since then so much more has been added! While working in a project management capacity for a software company, I got an up-close look at the way web work is done in the modern age of HTML5, CSS3, and cool JavaScript interactivity.

I started the Khan Academy Intro to JS course months back out of curiosity, but my work schedule caused it to fall by the wayside. After some developments at that company, I suddenly had some time to sit down and learn! In the Khan HTML course, one of the assignements was to make an HTML blog format. I thought that continuing the may be a good way to improve my web design skills. I've been experimenting with CSS and JS, and I plan to continue implementing new design updates to this page.
